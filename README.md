# Arcanea

## Setup Project

### Back

````terminal
cd back
pnpm i
pnpm dev
docker compose up
````
The back run at `http://localhost:3000`

To access database admin run
````
cd back
npx prisma studio
````
Go to `http://localhost:5555`





### Front


````terminal
cd front
pnpm i
pnpm dev
````

The front run at `http://localhost:1234`
